## How to Build 
* **Prerequisite**
    * A python editor
    * Mention/Importing Datasets Correctly
        * #### Note: Datasets and project files are all in here. You just need to find out the same name correspond to project. 

* **Installation**
    * *Clone the repository* from Github to your local machine. Open a terminal and run the following command:
    ```bash
    git clone https://github.com/Ede-Rojin-Delibas/Machine-Learning.git
    ```
    * Navigate to the project folder:
    ```bash 
    cd your-python-repository
    ```

* **Install Dependecies**
    * They are some package/Modules that they need to install to run the project.
        * For this repository I used some popular modules for my ML projects. For instance,pandas, matplotlib, numpy, scikit-learn,stats... 
        * Use ***pip*** for installation.
